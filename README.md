# esp-idf-smtp

Lightweight SMTP client over [esp_tls](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/api-reference/protocols/esp_tls.html) for [esp-idf-svc](https://github.com/esp-rs/esp-idf-svc) in Rust.
